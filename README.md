## 🌟 The Universal Blueprint Standard

This repository contains the **Supreme Universal Blueprint** templates for all project types (Web App, E-commerce, Engine, API, etc.).

### 📂 Structure

- **`UNIVERSAL-BLUEPRINT.md`** → The master template. Copy this to your project root as `BLUEPRINT.md`.
- **`AGENTS.md`** → The CEO-level mandate file for AI autonomous control.
- **`01-strategy.md`** → Strategic planning template.
- **`02-architecture.md`** → System architecture template.
- **`05-infrastructure.md`** → Infrastructure & Docker template.
- **`22-docker-governance.md`** → Container naming & management rules.

### 🛠 How to Use

1. **Clone this repo:**
   ```bash
   git clone https://github.com/Delqhi/CODE-BLUEPRINTS.git
   ```

2. **Copy to your project:**
   ```bash
   cp CODE-BLUEPRINTS/UNIVERSAL-BLUEPRINT.md my-project/BLUEPRINT.md
   cp CODE-BLUEPRINTS/AGENTS.md my-project/AGENTS.md
   ```

3. **Fill in the blanks:**
   - Replace `[Project Name]` with your project name.
   - Define your specific architecture in the relevant sections.

## 🏆 Best Practices (February 2026)

- **AI-First:** All blueprints are designed for AI-driven development (Sisyphus, Cline, etc.).
- **Docker-Native:** Modular container architecture is mandatory.
